# Titanic EDA 

## Project Overview
This project performs Mini Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Objectives
- Clean the dataset by handling missing values.
- Create AgeGroup and FamilySize features.
- Analyze survival patterns using GroupBy.
- Visualize insights using bar plots, histogram, and heatmap.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Titanic Training Dataset (train.csv)

## Key Insights
- Children had the highest survival rate.
- Passengers from Cherbourg (Port C) showed the highest survival rate.
- Small families had better survival chances than large families.
- Most passengers were between 20 and 40 years of age.
